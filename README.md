# Social Pulse — Live Social Media Counter

A tablet-optimised display app that shows live social media follower counts for Instagram, YouTube, TikTok, and X (Twitter). Designed to sit on a counter, desk, or shelf and update automatically — no secondary displays or hardware required.

## Overview

Social Pulse gives businesses a beautiful, always-on follower counter that runs in any tablet browser. Choose between an all-platform grid view or a single-platform focus mode. Numbers are always shown in full with no rounding.

## Features

- Live follower counter for Instagram, YouTube, TikTok, and X (Twitter)
- All-platform grid view or single-platform full-screen mode
- Exact follower counts — no rounding or abbreviation
- Auto-updates every 60 seconds with a visual countdown
- Gain/loss indicator showing changes since the last update
- Configurable handles and follower counts via built-in settings panel
- Real-time clock display
- Works on any tablet browser — no installation required
- Add to Home Screen on iPad or Android for a native app feel

## How to Use

1. Open the app URL in Safari (iPad) or Chrome (Android tablet)
2. Tap **Configure** to enter your social media handles and current follower counts
3. Choose **All Platforms** to display all four counters, or **Single Platform** to focus on one
4. Save — the display updates immediately and refreshes every 60 seconds

**To install as a home screen app on iPad:** Safari → Share → Add to Home Screen

**To install as a home screen app on Android:** Chrome → Menu (⋮) → Add to Home Screen
